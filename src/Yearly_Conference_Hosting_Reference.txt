# Conference Hosting Reference — Comprehensive Year-by-Year Map
# Update this reference to map on which platform a conference is hosted on each year
# We would like to use OpenReview (OR) as the primary paper retrieval source
# Routing to the corresponding proceedings if a conference is not hosted on OR.
# Based on exhaustive web search of OpenReview group pages, API docs,
# official Call for Papers, and DBLP/proceedings sites.
# Last verified: March 2026
#
# Legend:
#   OR  = OpenReview (primary submission/review system)
#   OJS = AAAI OJS (ojs.aaai.org)
#   CMT = Microsoft CMT
#   PMLR = Proceedings of Machine Learning Research (proceedings.mlr.press)
#   NP  = NeurIPS Proceedings (proceedings.neurips.cc / papers.nips.cc)
#   ACL = ACL Anthology (aclanthology.org)
#   IJCAI-P = IJCAI Proceedings (ijcai.org/proceedings)
#   ARR = ACL Rolling Review (on OpenReview, but different group structure)

# ACL — ACL Anthology historically; ACL Rolling Review (ARR) on OR since 2021
# ═══════════════════════════════════════════════════════════════════════
#
# Year  Submission System    OpenReview Group ID                     Proceedings          Notes
# ----  ------------------   ------------------------------------    ---------            -----
# ≤2020 Softconf/START       Not on OR                               aclanthology.org     Traditional submission system
# 2021  ARR on OR            aclweb.org/ACL/ARR/2021/...             aclanthology.org     ARR launched mid-2021
# 2022  ARR on OR            aclweb.org/ACL/ARR/2022/{Month}         aclanthology.org     Rolling monthly cycles
# 2023  ARR on OR            aclweb.org/ACL/ARR/2023/{Month}         aclanthology.org
# 2024  ARR on OR            aclweb.org/ACL/ARR/2024/{Month}         aclanthology.org     ACL, EACL, NAACL, EMNLP all use ARR
#                            aclweb.org/ACL/2024/ARR_Commitment                           Commitment venue for accepted papers
#                            aclweb.org/ACL/2024/Conference                               Direct conference group
# 2025  ARR on OR            aclweb.org/ACL/ARR/2025/{Month}         aclanthology.org
#                            aclweb.org/ACL/2025/Conference
#
# Complexity: ACL uses rolling review (ARR). Papers submit to ARR monthly,
# then "commit" to a specific conference (ACL, NAACL, EMNLP, etc.).
# The commitment venue ID is: aclweb.org/ACL/{year}/ARR_Commitment
# The conference group is: aclweb.org/ACL/{year}/Conference
#
# Fallback: aclanthology.org (all years, definitive proceedings)
#

#
# ICML:
#   {conf}.cc/{year}/Conference              → 2023+
#   (skip OpenReview for ≤2022, go straight to PMLR)
#
# NeurIPS:
#   NeurIPS.cc/{year}/Conference             → 2021+
#   (skip OpenReview for ≤2020, go straight to proceedings.neurips.cc)
#   NIPS.cc/{year}/...                       → only workshops for ≤2017
#
# AAAI:
#   AAAI.org/{year}/Conference               → 2025+
#   (skip OpenReview for ≤2024, go straight to OJS)
#
# ACL:
#   aclweb.org/ACL/{year}/Conference         → 2021+ (conference group)
#   aclweb.org/ACL/{year}/ARR_Commitment     → 2024+ (commitment venue)
#   aclweb.org/ACL/ARR/{year}/{Month}        → 2021+ (ARR rolling review)
#   (skip OpenReview for ≤2020, go straight to ACL Anthology)
#
# IJCAI:
#   NOT ON OPENREVIEW for main track (any year)
#   Go straight to ijcai.org/proceedings

# ═══════════════════════════════════════════════════════════════════════
# INVITATION SUFFIX patterns (in order of priority)
# ═══════════════════════════════════════════════════════════════════════
#
# Modern (2018+):
#   Blind_Submission     → ICLR 2018-2022, NeurIPS 2021-2022
#   Submission           → ICLR 2023+, ICML 2023+, NeurIPS 2023+, AAAI 2025+
#   Submitted            → rare alternate
#
# Legacy (≤2017):
#   submission           → ICLR 2013-2017 (lowercase!)
#   blind_submission     → possible variant
#
# Very early ICLR (2013-2014) used non-standard paths:
#   ICLR.cc/2013/-/submission                (no conference sub-group)
#   ICLR.cc/2014/-/submission/conference     (slash-separated variant)
#   ICLR.cc/2014/-/submission/workshop
